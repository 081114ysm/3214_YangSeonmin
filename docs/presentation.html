Overview

왜 만들었는가?

개발 공부를 하면서 느꼈습니다.

강의는 유튜브에서,
시간 관리는 뽀모도로 앱에서,
질문은 커뮤니티에서 —

전부 따로따로.

그래서 만들었습니다.
학습에 필요한 모든 것을 하나에 담은 플랫폼.

문제

학습 도구가 여러 플랫폼에 분산되어 집중력과 효율이 떨어짐

해결

강의 + 타이머 + Q&A + 진도관리를 하나의 웹 서비스로 통합

대상

체계적으로 학습하고 싶은 개발자 및 학생

LEARN · 학습

마이페이지

수강 중/완료 강의 수, 평균 진행률, 강의별 진도 바를 한 화면에서 확인합니다.

LEARN · 학습

배운 코드를 언어별로 저장하고, 메모와 함께 한 번의 클릭으로 복사할 수 있습니다.

FOCUS · 집중

집중 타이머

실시간 타이머로 학습 시간을 기록하고 히스토리와 통계로 돌아볼 수 있습니다.

ACCOUNT · 계정

JWT 기반 회원가입·로그인과 대시보드 통계를 통해 개인화된 학습 환경을 제공합니다.

이메일 — 필수 + 형식 검증 (정규식)
비밀번호 — 필수 + 4자 이상
닉네임 — 필수 + 2자 이상
질문 제목 — 필수 + 2자 이상
질문/답변 내용 — 빈 값 불가
이메일 중복 시 409 에러 반환

에러 처리

모든 API에 try-catch 적용
400 입력오류 / 401 인증실패 / 404 없음
409 중복 / 500 서버에러

Headers: Content-Type, Authorization
Credentials: true

DevFocus6 / 9
Demo Scenario

시연 시나리오

1

회원가입 & 로그인

회원가입 → 빈 값/잘못된 이메일 입력 시 에러 메시지 확인 → 정상 가입 → 로그인 → 헤더에 닉네임 표시 확인

2

강의 탐색

강의 목록 → 카드 클릭 → 영상 목록 + 시간 표시 확인 → "강의 보기" 클릭

3

집중 타이머

"시작" 클릭 → 실시간 타이머 동작 → "종료" → 기록 히스토리에 추가 → 대시보드에서 통계 확인

4

Q&A 게시판

질문 작성 (제목 + 내용) → 목록에서 클릭 → 상세 내용 확인 → 답변 등록 → 답변 표시 확인

5

코드 스니펫 & 마이페이지

스니펫 작성 → 언어 필터 → 복사 → 마이페이지에서 수강 진행률 / 완료 강의 통계 확인
