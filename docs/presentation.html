Overview

왜 만들었는가?

개발 공부를 하면서 느꼈습니다.

강의는 유튜브에서,
시간 관리는 뽀모도로 앱에서,
질문은 커뮤니티에서 —

전부 따로따로.

그래서 만들었습니다.
학습에 필요한 모든 것을 하나에 담은 플랫폼.

🎯 문제

학습 도구가 여러 플랫폼에 분산되어 집중력과 효율이 떨어짐

💡 해결

강의 + 타이머 + Q&A + 진도관리를 하나의 웹 서비스로 통합

👤 대상

체계적으로 학습하고 싶은 개발자 및 학생

Validation & Security

밸리데이션 & 보안

이중 검증 (Client + Server)

✓ 이메일 — 필수 + 형식 검증 (정규식)
✓ 비밀번호 — 필수 + 4자 이상
✓ 닉네임 — 필수 + 2자 이상
✓ 질문 제목 — 필수 + 2자 이상
✓ 질문/답변 내용 — 빈 값 불가
✓ 이메일 중복 → 409 에러

에러 처리

모든 API에 try-catch 적용
400 입력오류 / 401 인증실패 / 404 없음
409 중복 / 500 서버에러

인증 & 보안

✓ 비밀번호 bcrypt 해시 저장
✓ JWT 토큰 발급 (1시간 만료)
✓ Authorization 헤더 인증
✓ 인증 미들웨어 보호 API
✓ 미인증 시 로그인 페이지 리다이렉트

CORS 설정

✓ Origin: http://localhost:3000
✓ Methods: GET, POST, PATCH, DELETE
✓ Headers: Content-Type, Authorization
✓ Credentials: true

DevFocus5 / 8
Demo Scenario

시연 시나리오

1

회원가입 & 로그인

회원가입 → 빈 값/잘못된 이메일 입력 시 에러 메시지 확인 → 정상 가입 → 로그인 → 헤더에 닉네임 표시 확인

2

강의 탐색

강의 목록 → 카드 클릭 → 영상 목록 + 시간 표시 확인 → "강의 보기" 클릭

3

집중 타이머

"시작" 클릭 → 실시간 타이머 동작 → "종료" → 기록 히스토리에 추가 → 대시보드에서 통계 확인

4

Q&A 게시판

질문 작성 (제목 + 내용) → 목록에서 클릭 → 상세 내용 확인 → 답변 등록 → 답변 표시 확인
