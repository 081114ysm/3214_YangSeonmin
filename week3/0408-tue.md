## 오늘 공부 / 개발

강의 상세 페이지에서 "강의 보기" 버튼을 눌렀을 때 https://example.com/react-1 같은 더미 URL이 떠서 영상이 재생되지 않는 문제를 발견하였다.
schema.sql의 lessons INSERT 문에 들어 있던 더미 video_url을 모두 실제 YouTube 강의 영상 링크로 교체하였다.
또한 INSERT IGNORE로 인해 기존 DB에는 반영되지 않는 점을 고려해, 같은 파일 하단에 UPDATE 문을 추가하여 재실행 시에도 안전하게 덮어쓰도록 하였다.
mysql 클라이언트로 schema.sql을 다시 실행해 운영 DB의 lessons.video_url 9개를 모두 실제 영상으로 갱신하였다.

## ⚠️ 오류 / 어려웠던 점

처음에는 INSERT IGNORE만 수정해서 schema.sql을 다시 실행했지만, 기존 row가 이미 존재해 반영되지 않는 문제가 있었다.
INSERT는 그대로 두고 UPDATE 문을 추가하여 "신규 DB / 기존 DB" 양쪽 모두 동작하도록 정리하였다.
