# 4/6 (일)

## 오늘 공부 / 개발

프론트엔드에 /snippets 페이지를 만들고, 코드 스니펫을 카드 형태로 보여주도록 구현하였다.
언어별 필터 버튼, 새 스니펫 작성 모달, 삭제 버튼을 추가하였다.
또한 마이페이지(/mypage)를 새로 만들어 사용자가 좋아요한 강의, 수강 중인 강의, 저장한 스니펫 수 등을 한눈에 볼 수 있도록 구성하였다.

## ⚠️ 오류 / 어려웠던 점

코드 블록을 화면에 표시할 때 줄바꿈과 들여쓰기가 깨지는 문제가 있었다.
`<pre>` 태그와 whitespace-pre-wrap 클래스를 사용하여 원본 형식을 유지하도록 수정하였다.
