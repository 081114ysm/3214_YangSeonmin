# 4/4 (금)

## 오늘 공부 / 개발

강의 댓글 기능을 백엔드와 프론트엔드 모두 구현하였다.
댓글 작성 시 nickname을 join으로 함께 가져오도록 쿼리를 작성하고, 강의 상세 페이지 하단에 댓글 입력창과 목록 UI를 추가하였다.
로그인하지 않은 사용자는 입력창이 비활성화되고, 댓글 작성 버튼을 누르면 로그인 페이지로 이동하도록 처리하였다.

## ⚠️ 오류 / 어려웠던 점

댓글 작성 후 목록을 다시 fetch 하지 않고도 즉시 반영되도록 만드는 부분에서 시간이 걸렸다.
서버에서 댓글 작성 후 갱신된 전체 댓글 배열을 응답으로 내려주는 방식으로 통일하여 해결하였다.
