# 4/2 (수)

## 오늘 공부 / 개발

2주차에 만든 강의 상세 페이지에 사용자 상호작용 기능을 추가하기 시작했다.
courses 컨트롤러에 좋아요(like) 기능을 구현하고, course_likes 테이블을 통해 사용자별 좋아요 상태를 관리하도록 하였다.
프론트엔드의 강의 상세 페이지에 좋아요 버튼을 추가하고, 토글 시 likeCount가 즉시 갱신되도록 구성하였다.

## ⚠️ 오류 / 어려웠던 점

좋아요 토글 시 상태가 한 박자 늦게 반영되는 문제가 있었다.
서버에서 갱신된 likeCount와 liked 값을 함께 응답으로 내려주고, setState로 한 번에 반영하여 해결하였다.
