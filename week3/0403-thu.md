# 4/3 (목)

## 오늘 공부 / 개발

수강 신청(enroll) 기능을 구현하였다.
enrollments 테이블을 통해 어떤 사용자가 어떤 강의를 신청했는지 관리하고, 강의 상세 페이지에서 "수강 신청" → "수강 중" 상태로 즉시 전환되도록 구성하였다.
또한 강의 댓글 기능을 위한 course_comments 테이블을 설계하고 API 구조를 잡았다.

## ⚠️ 오류 / 어려웠던 점

이미 수강 중인 강의를 다시 신청할 때 중복 row가 들어가는 문제가 있었다.
INSERT IGNORE와 UNIQUE 제약을 함께 적용하여 해결하였다.
